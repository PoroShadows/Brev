Brev

A basic event system

#

API

  • bus.on(eventName, handler)
  • bus.once(eventName, handler)
  • bus.many(eventName, timesAvailable, handler)
  • bus.off(eventName, handler)
  • bus.trigger(eventName[, event])
  • bus.mixin(obj)
  • bus.create()
  • bus.reflect(eventName)

bus.on(eventName, handler)

  • eventName <String> The event name
  • handler <Function> The function handling the event
  • Returns: <brev> The instance

Registers a handler to the given eventName.

function handler(e) {}
bus.on('connect', handler);

bus.once(eventName, handler)

  • eventName <String> The event name
  • handler <Function> The function handling the event
  • Returns: <brev> The instance

Registers a handler to the given eventName. It will only be called one time before it is unregistered.

function handler(e) {}
bus.once('connect', handler);

bus.many(eventName, timesAvailable, handler)

  • eventName <String> The event name
  • timesAvailable <Number> The event name
  • handler <Function> The function handling the event
  • Returns: <Brev> The instance

Registers a handler to the given eventName. It will only be called x amount of times before it is unregistered.

function handler(e) {}
bus.many('connect', 3, handler);

bus.off(eventName, handler)

  • eventName <String> The event name
  • handler <Function> The function handling the event
  • Returns: <Brev> The instance

Unregisters a handler to the given eventName.

function handler(e) {}
bus.off('connect', handler)

bus.emit(eventName[, event])

  • eventName <String> The event name
  • event <Any> Any kind of value you want to pass to the handler(s) Default: undefined

Trigger a event to all handlers registered under the given event name.

bus.trigger('connect', { status: 'ok' })

bus.mixin(obj)

  • obj <Object> | <Any> The object you want the event system to mix into

Mixin the current event system with th specified object.

(This method will override any and all functions with the same names as the api.)

bus.mixin(myObj)

bus.createBus()

  • Returns: <Brev> A new event system

Creates a new and fresh event system.

var bus = brev.create()

bus.reflect(eventName)

  • Returns: <Object>
    • name: <String> The name of the event
    • exists: <Boolean> If there are handlers for the event
    • length: <Number> The number of handlers registered on the event
    • handlers: <Array.<Handler>> The handlers registered on the event

Get information about a specific event

var reflection = bus.create()
